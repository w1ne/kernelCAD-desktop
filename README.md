# kernelCAD Desktop

Native parametric CAD application for Windows, macOS, and Linux.

Companion to [kernelCAD-web](https://github.com/w1ne/kernelCAD-web) — runs OCCT natively for full performance without WASM overhead.

## Architecture

- **B-Rep kernel**: OpenCASCADE (OCCT) — native, no WASM
- **UI**: TBD (Qt / Dear ImGui / Tauri)
- **Language**: TBD (C++ / Rust)
- **File format**: `.kcd` (shared with kernelCAD-web)
